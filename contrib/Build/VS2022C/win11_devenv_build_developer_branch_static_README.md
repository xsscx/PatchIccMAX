
# Win11 Devenv Build Developer Branch Static

## Overview
This PowerShell script is designed for building the developer branch for a Windows 11 dev environment. It automates tasks related to developer branch build, ensuring an efficient workflow.

## Usage
To use this script, open a PowerShell window and execute the following command:

```powershell
.\win11_devenv_build_developer_branch_static.ps1
```

## Parameters
No parameters required.

## Requirements
- Windows PowerShell
- None

## Notes
Ensure you are on the correct branch before building
