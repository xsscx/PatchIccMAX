
# Win11 Devenv Build Pcs Refactor Branch Static

## Overview
This PowerShell script is designed for building the PCS Refactor branch for a Windows 11 dev environment. It automates tasks related to branch build, ensuring an efficient workflow.

## Usage
To use this script, open a PowerShell window and execute the following command:

```powershell
.\win11_devenv_build_PCS_Refactor_branch_static.ps1
```

## Parameters
No parameters required.

## Requirements
- Windows PowerShell
- None

## Notes
Ensure refactor branch is ready and all dependencies installed
