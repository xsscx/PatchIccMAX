
# Build Revert Master Branch

## Overview
This PowerShell script is designed for reverting the master branch. It automates tasks related to git branch management, ensuring an efficient workflow.

## Usage
To use this script, open a PowerShell window and execute the following command:

```powershell
.\build_revert_master_branch.ps1
```

## Parameters
No parameters required.

## Requirements
- Windows PowerShell
- None

## Notes
Backup current branch before proceeding
