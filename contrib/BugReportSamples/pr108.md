### Build Tools/CmdLine/IccApplyNamedCmm

```

cmake -DCMAKE_INSTALL_PREFIX=$HOME/.local -DCMAKE_BUILD_TYPE=Release . && make -j10  VERBOSE=1
```

```
./iccApplyNamedCmm

```
