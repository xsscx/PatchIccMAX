
# Static Link Branch for Windows

Hello and Welcome to Hoyt's Static Link Branch for Windows of the DemoIccMAX Project. Open Developer Powershell and Paste in the following command to Build:

```
iex (iwr -Uri "https://raw.githubusercontent.com/xsscx/PatchIccMAX/refs/heads/development/contrib/Build/VS2022E/static_build_cli_production.ps1").Content
```
